# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Yes             |

## Reporting a Vulnerability

PysternBlot takes security seriously. If you discover a security vulnerability, 
please do **not** open a public GitHub issue.

Instead, please use GitHub's private vulnerability reporting feature:

👉 [Report a vulnerability](https://github.com/TheBrickalista/PysternBlot/security/advisories/new)

You can expect:
- Acknowledgement within 48 hours
- A status update within 7 days
- Credit in the release notes if you wish

## Scope

This policy covers the PysternBlot application and its dependencies as 
distributed via:
- PyPI (`pip install pysternblot`)
- GitHub Releases (macOS `.app` and Windows `.exe`)

## Out of Scope

- Vulnerabilities in third-party dependencies should be reported to their 
  respective maintainers
- Issues already publicly disclosed
